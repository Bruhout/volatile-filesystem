### Volatile Filesystem

Volatile filesystem is a wordplay on Virtual FileSystem, which share the same acronym VFS. It is a simulation of a Inode based on-disk filesystem written inside volatile memory, for an Operating Systems class.

Build requirements:
1. C standard library
2. Compiler with support for C99
3. Make build system (optional)

Build instructions:
1. Compile `main.c` with `-std=c99`.
2. Alterntively, use the provided makefile, run `make` to build and `make run` to start the program.
